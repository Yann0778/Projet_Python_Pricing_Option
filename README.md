# Projet_Python_Pricing_Option
Étude comparative du pricing d'options sur AAPL — Black-Scholes vs Random Forest vs MLP. Les modèles ML apprennent à prédire le mid-price (bid+ask)/2 directement depuis les données de marché, sans hypothèse sur la dynamique du sous-jacent. GridSearchCV, cross-validation 5-fold, analyse des Grecques et tableau de synthèse automatique.
